
# Spawnify Games Official Website

The official website for **Spawnify Games**, a Roblox development group creating fun, original, and creative gaming experiences.

## What is Spawnify Games?

Spawnify Games is a Roblox development group dedicated to turning creative ideas into playable worlds. We focus on innovation, teamwork, and skill development in game design, scripting, building, and UI design.

**Tagline:** *where every idea spawns*

## About This Repository

This is a fork of the [official Spawnify Games website](https://github.com/spawnifygames/spawnifygames.github.io) hosted at **spawnifygames.win**. The repository contains the static HTML and image assets for the public-facing website showcasing the group's games and team information.

## Features

- **Game Showcase**: Display of Spawnify Games' published Roblox games
- **Team Roles**: Overview of different roles in the development group (Owners, Developers, Builders, UI/Designers, Testers, Community)
- **Community Links**: Quick access to join the group across platforms:
  - Roblox Group
  - Discord Server
  - YouTube Channel
- **Responsive Design**: Mobile-friendly layout with modern styling
- **Accessible**: Skip-to-content link and semantic HTML structure

## Games Featured

1. **Race and Race** — Drive fast and beat the clock in an exciting racing game
2. **Islands of Work and Wonder** — Work, sell loot, and explore across multiple islands
3. **The Bob [Horror]** — Survival horror game where you must find safety and survive
4. **Tower of Heights** — Climb to the top of the tower and choose your next path
5. **Five Nights at Klavs** — FNAF-inspired parody game with a 5-night challenge and bonus shift

## Quick Start

### Local Preview

1. Clone the repository:
   ```bash
   git clone https://github.com/spawnifygames.github.io.git
   cd spawnifygames.github.io
   ```

2. Open `index.html` in your web browser to preview the site locally.
